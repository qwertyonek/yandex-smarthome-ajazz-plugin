Yandex OAuth токен
Локальная утилита для получения отладочного access‑token Яндекса. Данные никуда не отправляются кроме официальной страницы OAuth.
Убедись, что в настройках приложения Redirect URI совпадает с адресом, по которому открыт этот файл.
Токен извлечён из URL фрагмента после авторизации.
Храни токен в секрете. Не передавай его третьим лицам.
